# Автоматизация ИИ-сводок

Эта папка содержит новый тестовый контур автоматизации.

## Важно

Контур не должен изменять или публиковать файлы из папки `/posts/`,
пока RSS проходит проверку в Дзене.

Текущая рабочая публикация:

- https://rybalka.one/posts/
- https://rybalka.one/posts/rss.xml

## Тестовый контур

Все результаты тестовой сборки должны сохраняться только в:

- `automation/archive/`
- `automation/content/`
- `automation/preview/`

До отдельного разрешения запрещены:

- FTP-деплой;
- изменение папки `/posts/`;
- изменение рабочего `rss.xml`;
- запуск по расписанию;
- автоматический commit в ветку `main`.
